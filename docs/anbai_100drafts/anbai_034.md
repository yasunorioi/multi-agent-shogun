# 【学会ポスター】Anbai理論：深夜に発見した真実を学術的に発表しようとするポスター

**塩見あんばい**（AIコンサルタント / 独立行政法人 最適化しすぎ研究機構 / このポスターは深夜3時に完成した）

---

## ■ 背景（眠いけど大事なことを言う）

えっと。

某クライアントとの環境制御PJ、NDA締結してます。詳細は言えない。でも114日間、32,470行のデータを取った。これは本当のこと。

で、LLMを毎時呼んでたら全体の29.1%が緊急になったんですよ。

29.1%よ。

全部で100%のうち、29.1%が緊急。

……これ、多くないですか。多い。絶対多い。普通じゃない。

Goodhartさんが1975年に「指標が目標になったらダメになる」って言ってた。これがそれ。そういうことらしい。正直、このポスター作り始めるまでちゃんと理解してなかったけど、今は分かる気がする。

## ■ Herbert Simonの話（重要）

1956年にHerbert Simonという人が「satisficing（満足化）」を提唱した。

「最適解を求めるな、十分よい解で止まれ」

これ、70年前の話。

私は2026年に、32,470行のデータを使って同じことを発見した。

……遅くないですか。遅い。でも発見できたからいいと思う。

眠いけど、これは大事なことだから言っておく。

「止まれ」

それだけでいい。
