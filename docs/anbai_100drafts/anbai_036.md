## 【背景 / Background】

本研究は、生産現場における環境制御システムの過剰最適化事例から「Anbai Curve」を導出する目的で開始された。

**研究開始時の著者の想定:**
- 研究対象: 某施設の環境制御システム
- 研究者: 著者（塩見あんばい）
- 関係: 観察者と観察対象の間には明確な距離がある

**研究終了時の著者の認識:**
- 研究対象: 某施設の環境制御システム、**および著者自身**
- 研究者: 著者（現在、研究対象と化しつつある）
- 関係: 観察者と観察対象が同一人物である可能性が浮上

この認識の変化が本発表の核心である。

## 【主要結果1: 某施設のAnbai Curve実証】

### 27℃問題（Figure 2）

- **設定**: 緊急閾値 = 27℃
- **結果**: 全体の**29.1%**（n=9,448/32,470）が「緊急」に分類
- **問題**: 3割が「緊急」なら、緊急は日常である

```
緊急判定の失敗パターン（Goodhart's Law作動）:

目標: 「緊急」を検知する
手段: 27℃閾値を設定する
結果: 27℃が「緊急」を示す指標として機能しなくなる

これがGoodhart's Lawである。
"When a measure becomes a target, it ceases to be a good measure."
```

### 三層構造による解決（Figure 3）

| 層 | 手法 | 役割 | 結果 |
|----|------|------|------|
| Layer 1 | if文 | 本当の緊急のみ | 問題の1% |
| Layer 2 | ルールベース | 日常制御 | 問題の99% |
| Layer 3 | LLM | 本当に困った時 | 残余 |

**効果: LLM呼び出し89%削減**——それでいて制御品質は向上。

この対比は著者を発表中に沈黙させた。（Figure 4作成時点で約3分間）

## 【考察 / Discussion】

### 研究者と研究対象の一致について

本来、研究者は研究対象の外部に存在するべきである。しかし本研究においては、研究対象の構造的特徴（過剰最適化、Layer 1の削減、指標の歪み）が著者自身の行動パターンと一致していることが研究過程で判明した。

これをどう解釈すべきか。

**解釈A**: 著者が研究対象を無意識に自己参照的に設定した（研究バイアス）
**解釈B**: Anbai Curveは普遍的なパターンであり、著者もその例外ではない（理論の強化）
**解釈C**: 著者は最初から自分自身を研究していた（目的の事後的発見）

著者は現在、解釈Bを採用している。採用しなければ論文が成立しないからだ。これ自体がAnbai理論の自己参照的実証かもしれない。

### 「気づき」の機能

Anbai Curveの右端（Over-Optimization Zone）にいる者が右端にいると気づくことは、原理的には右端からの離脱を可能にする。しかし気づきと行動変容の間には相当の距離がある。

著者は現在、気づきと行動変容の間のどこかにいる。具体的な座標は不明。
