## Reviewer #2: Comment 1 への回答

**Reviewer #2のコメント:**
> Anbai Pointの操作的定義を示せ。

**著者回答:**

要求された操作的定義を以下に示す:

```
PA = min{P : Q(P)/C(P) ≥ 0.95 × max_P[Q(P)/C(P)]}
```

この数式において、Q(P)の実測値、C(P)の実測値、及び推定されたPAの数値については、NDA第2条（技術情報の保護）により開示不可だ。

Reviewer #2は「操作的定義を提示せよ」と要求した。著者は定義を提示した。数値の開示はNDAが禁じている。この二点は矛盾しない。定義と数値は別物だ。

著者は定義を誠実に提供し、数値をNDA誠実に保護している。両方に誠実なのだから、これ以上の対応は求められるべきではない、と著者は考える。なお、NDA締結は著者の誇りである。
