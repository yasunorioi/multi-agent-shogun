## 後輩へのプレアンブル

この文書を読んでいる君が、もし初めて研究論文の査読を受けた若いエンジニアや研究者なら——ぜひ続きを読んでほしい。

査読は怖い。私も最初は怖かった。見知らぬ誰かが匿名で君の仕事を批判する。しかしそのプロセスを経ることで、研究は良くなる。

Anbai理論を査読に適用すれば: 査読への対応も「ちょうどいい」が最善だ。過剰に防衛的になっても、過剰に従順になっても駄目だ。

では実例を示す。

## Reviewer #1: Comment 1 への回答

**Reviewer #1のコメント:**
> Simonの満足化理論の再記述ではないか。独自性を明示せよ。

**著者回答（並びに後輩への解説）:**

これは「独自性の疑義」コメントだ。最も多いタイプの査読コメントの一つ。

**対応の鉄則: 相手が指摘した具体的な点に具体的に返答せよ。感情的に反発するな。**

著者の回答:

本研究の独自性は三点ある。第一に、Simonのsatisficingを現代AIシステムの設計原則として数式化した（PA = argmax Q(P)/C(P)）。第二に、Goodhart's Lawの「作動タイミング」を実測した（27℃問題: 29.1%の過剰判定）。第三に、理論から設計変更（三層構造化）を経て89%削減という定量的結果を示した。

「先人の肩の上に立つ」ことは研究の常だ。重要なのは「どの高さで立ったか」と「そこから何を見たか」だ。

**後輩へ:** この返答では防衛していない。代わりに、「貢献の地図」を描いて見せた。貢献の地図を描けば、査読者は自分の批判の場所を特定できる。それで批判が正確になる。批判が正確になれば、研究が良くなる。

## Reviewer #1: Comment 2 への回答

**Reviewer #1のコメント:**
> 単一施設データの外部妥当性の問題。

**著者回答（並びに後輩への解説）:**

**対応の鉄則: 正当な指摘は素直に認めよ。ただし認めることと「無価値」は別だ。**

著者の回答:

外部妥当性が限定的であることは事実だ。改訂版でタイトルを「事例分析」に変更し、この限界を明示した。

同時に、単一施設のn=32,470レコードという内部的な豊富さは、「当該施設においては」という限定付きで価値のある知見を提供する。外部妥当性の低さは「価値がない」を意味しない。それは「適用範囲が限定される」を意味するだけだ。

**後輩へ:** 「限界を認める」ことと「研究価値を否定する」ことを混同するな。査読者も、多くの場合は後者を意図していない。謙虚さと自信は両立する。謙虚さ: 限界を認める。自信: 限界の中での価値を主張する。

## Reviewer #2: Comment 1 への回答

**Reviewer #2のコメント:**
> Anbai Pointの操作的定義を示せ。

**著者回答（並びに後輩への解説）:**

**対応の鉄則: 「定義できない」ではなく「定義しなかった」ことへの対応と「定義した上での限界」を区別せよ。**

著者の回答:

要求に応じ、操作的定義を追加した:

```
PA = min{P : Q(P)/C(P) ≥ (1-ε) × max_P[Q(P)/C(P)]}
（ε=0.05）
```

同時に、この定義の適用にはQ(P)とC(P)の実測が必要であり、本事例では全点での測定は困難だったことを明示した。

**後輩へ:** 査読者が「定義を出せ」と言ったとき、「出せない」で終わる人と「出して、その限界も示す」人がいる。後者が正しい。定義の不完全さを認めることは弱さではなく、正確さだ。不完全な定義は、定義なしより常に良い。

## Reviewer #2: Comment 2 への回答

**Reviewer #2のコメント:**
> 利益相反の開示。

**著者回答（並びに後輩への解説）:**

**対応の鉄則: 利益相反は早く・明確に・詳細に開示せよ。隠す誘惑に負けるな。**

著者の回答:

著者は当該施設のシステム設計に関与しており、設計の有効性を示すことに動機を持つ可能性がある。この利益相反を論文冒頭に明示した。

**後輩へ:** これは特に伝えたい。利益相反の開示は「弱点の露出」ではない。「誠実さの証明」だ。

業界で働いていれば、自分が設計したものを評価する機会は必ず来る。その時、利益相反を隠すことは短期的に楽だが、長期的に信頼を失う。最初から「私はここに動機を持っている」と言える人間が、長く信頼される。

著者は経験から学んだ。君は著者の経験から学んでくれ。

## Reviewer #3: Comment 2 への回答

**Reviewer #3のコメント:**
> 架空文献は意図的か。

**著者回答（並びに後輩への解説）:**

意図的だ。本論文は4月1日発表の特殊論文であり、架空と実在の混在は研究設計の一部だ。改訂版でラベルを付与した。

**後輩へ:** 普通の研究論文に架空文献を入れるな（これは言うまでもないが念のため）。

本論文の場合、「嘘の日に真実を出す」というコンセプトがあるから成立している。コンテキストなしに架空文献を入れたら、それは単純な研究倫理違反だ。

型破りと無節操は違う。型破りには意図が必要だ。意図のない型破りは単なる間違いだ。
