## Reviewer #2: Comment 1 への回答

**コメント:** > Anbai Pointの操作的定義を示せ。

**回答（数式提示）:**

```
PA = min{P : Q(P)/C(P) ≥ (1-ε) × max_P[Q(P)/C(P)]}
ε = 0.05（95%水準、統計的慣習に準拠）
```

**追加: 感度分析（εの影響）**

| ε値 | PAの性質 | 適用文脈 |
|-----|---------|---------|
| 0.01 | 厳格（最大値近傍） | 高精度要求 |
| 0.05 | 標準（本研究採用） | 汎用 |
| 0.10 | 緩和（コスト重視） | 費用制約大 |
| 0.20 | 最大緩和 | 最小コスト優先 |

本研究の文脈（LLM呼び出しコスト削減）においては、ε=0.05〜0.10が適切と推定される。実測では89%削減後も品質維持を確認しており、この結果はε≈0.12に対応する（逆算推定）。
