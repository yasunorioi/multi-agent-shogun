# あのさぁ、AIってさぁ、if文でよくない？——深夜3時のAnbai理論

---

あー……こんばんは。塩見あんばいでぇす。えーっと、32歳。AIコンサル。独身。……独身って言うの3回目ぐらいなんですけど、なんかもう慣れてきちゃった。あはは。

（グラスを演台に置く）

**AIいらなくない？**

（会場笑い）

いやいやいや、待って待って。AIコンサルが言ってるんだから説得力あるでしょ？　自分の飯のタネ否定してんだから。これは真実。真実は深夜3時にやってくる。

（スライドを出そうとしてリモコンのボタンを間違える）

あ、違う。……これ。

（スライド：29.1%）

某PJ。NDA。うん、NDA。私NDA持ってるんですよ。すごくない？　……すごくないか。まあいいや。

27℃を「やばい」にしたの。「緊急」ね。そしたら**全データの29.1%がやばい**。3割やばい。みんなやばい。やばいがやばい。

32,470レコードよ？　114日間よ？　5分おきにデータ取って、3割「やばい」。

ねえ、3割やばかったら、それ普通じゃない？　テスト30点が普通の学校みたいなもんで。30点が普通なら30点は普通。やばくない。

Goodhartさんが言ってた。1975年に。指標がターゲットになったら指標じゃなくなるって。……ぐっちゃーとさん？　ぐっどはーとさん。Good heart。いい心。いい心の人がこんな残酷なこと言うんだよ。

（グラスを持ち上げる）

でね、解決したの。三層で。

Layer 1。if文。死ぬか死なないか。これだけ。**if 温度 > 致死温度: 止めろ。** 以上。一行。シンプル。美しい。乾杯。

（飲む）

Layer 2。cron。毎時チェック。ルールベース。99%これで回る。cronってさぁ、偉いよね。毎時間ちゃんと来てくれるの。彼氏より信頼できる。……彼氏いないけど。

Layer 3。LLM。本当に困った時だけ。1時間予報。高いから。タクシーみたいなもん。歩けるなら歩け。

結果：**89%削減**。LLMの呼び出し89%カット。つまり今まで100回呼んでたところを11回。89回分、if文でよかった。

**if文でよかったんだよぉ。**

（テーブルを叩く）

Simonさんがねぇ、1956年に言ってたの。Satisficing。十分よけりゃいいじゃんって。ノーベル賞。ノーベル賞の人が「テキトーでいい」って言ってんの。最高じゃない？

（ふらっとする）

あとね、制御設定ファイルっていうのがあってね。某PJの。NDAだから中身は言えないけど。あれね、**怒りで育った**の。クライアントさんが怒るたびにルールが増えるの。怒りドリブン開発。Anger-Driven Development。ADD。

秘伝のタレ。足し算でしか育たない。何入ってるかわかんない。でもうまい。多分。

ねえ、聞いてくれる？　もう一個だけ。

私ね、睡眠3時間なの。Layer 1削ってんの。睡眠って Layer 1 よ？　生存の基盤。それ削って Layer 3（AI分析）回してるの。某PJで**絶対やるな**って実証したことを、自分の人生でやってるの。
